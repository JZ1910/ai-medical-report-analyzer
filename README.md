# AI Medical Report Analyzer

An AI-powered application that processes medical reports and extracts meaningful insights using Retrieval-Augmented Generation (RAG).

## Key Features
- Upload and analyze medical reports (PDF/Text)
- Convert text into embeddings
- Retrieve relevant medical context using vector similarity search
- Generate insights using LLMs

## Tech Stack
- Python
- Streamlit
- Sentence Transformers
- OpenAI API
- Pandas

## Overview
This project leverages Retrieval-Augmented Generation to analyze medical data and provide context-aware insights. It demonstrates how AI can assist in interpreting complex medical information efficiently and accurately.
